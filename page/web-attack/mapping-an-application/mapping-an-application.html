Mapping An Application

Step 1 - Scraping the Application

Gather as much information about the application as possible. This includes:

  • Application Pages and Functional Paths
  • Every Link
  • robots.txt and other exposed resources
  • Identify Input Vectors:
    -URL parameters
    -POST body (form or JSON)
    -Headers (like X-User-ID, Referer)
    -Cookies
    -WebSocket messages (if used)
